#Prodesk IT Website
Hi there! 
This project is a simple and modern website I built for a digital marketing company called **Prodesk IT**. The goal was to create a clean, responsive, and user-friendly interface using Tailwind CSS and a bit of JavaScript.

---

## What this project does
This website includes:
* A **beautiful landing page** with a gradient background
* A **services section** to showcase offerings like SEO, Web Development, and Marketing
* An **about section** describing the company
* A **contact form UI**
* A **mobile-friendly navigation menu**
* A **dark mode toggle**  for better user experience

---

##Built with
I used simple and powerful tools:
* HTML
* Tailwind CSS
* JavaScript

##How to run it

You don’t need anything fancy to run this project:
1. Download or clone the repository
2. Open the folder
3. Double-click on `index.html`
That’s it — the website will open in your browser 🎉

---

##Dark mode
There’s a dark mode feature included.
Just click the  button in the navbar to switch between light and dark themes.
